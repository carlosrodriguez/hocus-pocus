Defaults

Mixins

There is a font-size mixin which allows to set the element's font size and keep the proper vertical rhythm:

div
  @include font-size(12)

// div
//   font-size: 0.75rem;
//   line-height: 1.0625rem;

Functions

Sizing functions return a quotient/product of the provided value.

div
  padding: quarter(12px)
  // padding: 3px

div
  padding: third(12px)
  // padding: 4px

div
  padding: halve(12px)
  // padding: 6px

div
  padding: double(12px)
  // padding: 24px

div
  padding: triple(12px)
  // padding: 36px

div
  padding: quadruple(12px)
  // padding: 48px
