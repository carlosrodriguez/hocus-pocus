Media

Media object works with any html element.

Gutter size modifiers are generated for each size in $spacing-map.

Usage:

Example Text

<div class="media">
  <img src="...">
  <div class="media-body">
    ...
  </div>
</div>

Example Text

<div class="media media-middle">
  <img src="...">
  <div class="media-body">
    ...
  </div>
</div>
