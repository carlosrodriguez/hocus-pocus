Hocus-Pocus is a design-free Sass framework inspired by Primer and inuitcss.

The main idea behind is to make a universal and lightweight stylesheets starter kit that only includes the most common stuff. There is no need to overwrite existing Hocus-Pocus classes because every default can be modified in a clean way by changing a variable. I wrote a short story why have I started working on another framework.

Included features:

  • Enhanced normalize.css reset
  • Default colors set and color helpers (thanks to clrs.cc)
  • Global box-sizing set to border-box
  • Flexbox based grid
  • Fraction width helpers
  • Responsive web design support with customizable breakpoints and approach (mobile or desktop first)
  • Visibility and spacing helpers (generated for specified responsive breakpoints!)
  • Base typography with vertical rhythm (based on Gridlover)
  • Lists, definition lists and menus
  • Image helpers
  • Tables
  • Forms
  • Buttons
  • Media object
  • Boxes
  • Navbar
  • Hero
  • Sticky footer
  • Clearfix
